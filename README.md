# Pizza Sales Analysis using SQL

## Project Overview
This project analyzes pizza sales data using SQL to uncover business insights related to revenue, customer preferences, and sales trends. It demonstrates SQL querying skills and data-driven decision making.

---

## Objectives
- Analyze overall sales performance
- Identify top-selling and least-selling pizzas
- Calculate revenue metrics
- Explore customer ordering patterns
- Generate actionable business insights

---

## Tools Used
- SQL
- MySQL
- GitHub

---

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions
- JOINS
- Subqueries

---

## Business Questions Answered
- What is the total revenue?
- Which pizzas generated the highest revenue?
- What are the top-selling pizza categories?
- Which pizza sizes are most popular?
- What are the peak order hours?
- What is the average order value?

---

## Key Insights
- Classic pizzas contributed the highest revenue.
- Large-sized pizzas were the most frequently ordered.
- Evening hours recorded the highest number of orders.
- A small number of pizzas generated a significant portion of total revenue.
